# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**Short version:** be kind, be constructive, assume good faith. Harassment, discrimination,
personal attacks, or sustained disruption are not tolerated — in issues, PRs, discussions,
or anywhere the project happens.

**Enforcement:** report unacceptable behavior privately to the maintainer (via GitHub).
Reports are handled confidentially. Consequences scale from a warning to a permanent ban,
at the maintainer's discretion, following the Covenant's enforcement guidelines.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
